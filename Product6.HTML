Examples of New Product Development

New Generation Automated Solvent Extraction

Key Contributions

Governance focus: Reviewed New product architecture; Design/GUI contributions; Identification of Key differentiators; FTO & IP filings; Initiated over communication to overcome communication barriers with the team in China

Challenge

Previous generation automated solvent extraction units were either sequential or manual and required a lot of time to process samples. For some applications the only control available for reducing the solvent uptake was by reducing the sample size via a smaller cell size. However this approach had impact on the detection limits and in some instances for compounds found in the environment the matrix effects mandated higher levels of samples. Additionally after a sequential process the sample extracts required evaporation or concentration and required additional equipment and added labor to process

Solution

Several solutions were explored to address the above challenges. A new mode of gas assisted extraction that lowered the solvent uptake and leverage the gas diffusion to extract the analytes and lowered the solvent usage under all sample size conditions. A new mode of pseudo parallel extraction was pursued to address the limitation of the sequential mode of extraction. The cost of implementation was reduced by pursuing the pseudo parallel in place of true parallel. An evaporation routine was added with end point detection using a camera interface and machine learning. Automation was conferred by the use of a gantry and a barcode based automated method installation was pursued.

Innovation

Several patented technologies are evident in the design. The key innovations are a) Parallel Extraction to expedite sample processing b) integrated evaporation to complete the workflow c) Automated level sensing via machine learning and auto sensing d) New gas assisted extraction reducing solvent usage e) Modern GUI with a barcode based deployment of methods

Value

The biggest value is the walkaway automation from loading the sample to collecting the concentrated extract after evaporation to the programmed level. Other values are apparent from the write up such as throughput, reduction in solvent usage, reducing errors and conferring full automation for sample preparation.
